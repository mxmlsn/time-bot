# üåç Telegram Time Bot (English Version)

A simple and handy bot for time conversion in Telegram chats. Perfect for distributed teams and friends located in different time zones.

### üöÄ How it works
The bot listens to messages. If you mention it or simply type the time with a city code, it automatically converts that time for **Moscow, Paris, Yerevan, and Buenos Aires**.

It also generates a **[+ add to calendar]** link, allowing you to instantly create a Google Calendar event for the calculated time.

### üìù Command Format and Error Resilience

The bot is designed to be highly **resilient** to typos and incorrect keyboard layouts. It recognizes several different characters for a single city, including Cyrillic, Latin, and adjacent symbols on the keyboard.

| City | Primary Letter | Allowed Aliases | Examples that will work |
| :---: | :---: | :---: | :--- |
| **Moscow** | –º | `m`, `v` | `15:30v`, `18 m` |
| **Paris** | –ø | `p`, `–∑` | `20:00–ø`, `12p`, `17:45–∑` |
| **Yerevan** | –µ | `e`, `y`, `t` | `10–µ`, `19:20e`, `09y`, `14t` |
| **Buenos Aires** | –± | `b`, `,` (comma) | `22–±`, `13:15b`, `11,` |

**Usage Examples:**
* `20 –º` (20:00 Moscow time)
* `18:30–ø` (18:30 Paris time)
* `@botname 14 –±` (14:00 Buenos Aires time)
